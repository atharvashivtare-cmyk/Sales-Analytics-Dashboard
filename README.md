📊 Sales Analytics Dashboard

A Sales Analytics Dashboard built with Python and Power BI to analyze sales performance, revenue trends, customer insights, and product performance. This project demonstrates data cleaning, analysis, and visualization skills using real-world sales data.

---

🚀 Features

- Sales Overview Dashboard
- Revenue Analysis
- Profit Analysis
- Category-wise Sales
- Region-wise Sales
- Monthly Sales Trends
- Top Selling Products
- Interactive Filters and Charts

---

🛠️ Technologies Used

- Python 3.12
- Pandas
- NumPy
- Matplotlib
- Power BI
- CSV Dataset

▶️ How to Run

1. Clone the repository.
2. Install the required libraries.

3. Open the Power BI dashboard or run the Python analysis scripts.
4. Explore the interactive dashboard.
